Functions in Python:
A function is a block of code that performs a specific task and can be reused multiple times.
Functions help make programs organized and reduce repetition.
Syntax:
def function_name():
    statements
function_name()
Types of Functions:
1. Built-in functions - print(), input(), len()
2. User-defined functions - created using def keyword
 Advantages of Functions:
1.Code reusability
2.Easy debugging
3.Better readability
4.Modular programming
