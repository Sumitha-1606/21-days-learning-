Variables and data types:
    Used to store data in memory
*In python
    No need to declare data type separately.
    Type is decided automatically.
Data Types in python:
    Data type specify the type of value a variable holds.
    python automatically identifies the data type based on the value aasigned.
Types of Data Types:
* Integer(int) - Used to store whole numbers.
* Float(float) - Used to store decimal numbers.
* String(str)  - Used to store text.
* Boolean(bool)- Used to store true or false values.
* type()       - Used to check data types.
