Introduction to Python:
Python is a high-level, interpreted, object-oriented programming language.
It was created by Guido van Rossum in 1991.
Python is simple, readable, and easy to learn. It is widely used in web development, data science, artificial intelligence, automation, and software development.
How to Print in Python:
In Python, output is displayed using the print() function.

The print() function is a built-in function used to display text or variables on the screen.print is a built-in function.
